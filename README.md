# Start Project
```
 composer install 
```

migration run 
```
php artisan migrate:refresh
```
Database Configration Edit in .env file

```
DB_CONNECTION=mysql
DB_HOST= < Your DB HOST >
DB_PORT= < Your DB PORT default 3306>
DB_DATABASE= < YOUR DB NAME >
DB_USERNAME= < YOUR DB USER > 
DB_PASSWORD= < YOUR DB PASS >
```

Run Server

```
php artisan serve 
```
